I would use unit testing to test my functions by inputting the correct 
parameters into the function definition, and outputting the contents to 
see if it was done correctly. I chose to create a function to print the 
main contents of the std::map, in my case, the farmMap. I would check the 
output against the file data it is read from, to make sure that the output 
is correct. I would also make small changes in the output formatting, 
like with the set widths, to make the data more readable.
